# Credit Card Financial Dashboard – Power BI Project

## Project Overview

This project presents an interactive **Credit Card Financial Dashboard** built using Power BI to analyze customer spending behavior, transaction trends, and overall credit card performance.

The dashboard provides insights into revenue, transaction volume, customer demographics, and weekly performance trends to support data-driven decision-making.

---

## Project Objective

To develop a comprehensive weekly credit card dashboard that provides insights into key performance metrics and trends, enabling stakeholders to monitor and analyze credit card operations effectively.

---

## Tools & Technologies Used

* Microsoft Power BI
* PostgreSQL
* SQL
* DAX
* Power Query
* Data Visualization

---

## Dashboard Features

* Revenue Analysis by Week
* Week-over-Week Revenue Comparison
* Revenue by Card Category
* Revenue by Income Group
* Revenue by Education Level
* Revenue by Customer Job
* Revenue by Marital Status
* Revenue by Age Group
* Top 5 States Analysis
* Customer Segmentation Analysis

---

## Key KPIs

* Total Revenue
* Total Interest Earned
* Total Transaction Amount
* Total Transaction Count
